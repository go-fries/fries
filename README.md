```bash

```

## Modules Snapshot

- `x/gantt/v3`: 渲染 mermaid Gantt（依赖/状态/里程碑/进度/日历排除、todayMarker/时区可控），示例见 `x/gantt/examples/full_mermaid.gantt` 与 `x/gantt/examples/basic/`.
